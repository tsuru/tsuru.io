Features

Run anything

Tsuru goes beyond 12 factor apps. Run any application written in any language or framework.

Deploy fast and safe

The deploy process is really simple and no special tools needed, just a simple git push.

Scale

Grow your application dynamically allocating resources with ease.

Optimize resources

Your application units are distributed between multiple nodes increasing resource utilization.

Failproof

Tsuru keeps your apps running by rebalancing resources and recovering failed units and nodes automatically.

Iaas integrated

Tsuru is integrated with EC2, Apache Cloudstack which allows auto provisioning simple, painless and fast.
